Welcome to N0CAPE. This utility dynamically extracts and pairs network interfaces with their respective switch hostnames from raw data.

> Instructions (01)

Provide the engine with raw data (Ticket descriptions, CDP/LLDP outputs, or Inventory tables). As long as the text contains a Switch Hostname (e.g., CO-ARN-CAS0302) and Port Numbers (e.g., gi1/0/3), the tool will map them.

Supported Formats & Auto-Correct: It supports single ports, comma-separated shorthand (Gi3/0/1,3,5), and automatically expands ranges like Gi2/0/23 - Gi2/0/27. It also features a typo engine that automatically corrects Gl (G-ell) inputs into valid Gi formats.
