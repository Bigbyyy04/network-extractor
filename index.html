N0CAPE // v2.0 [Automated Port Extractor]

Welcome to N0CAPE. This utility dynamically extracts and pairs network interfaces with their respective switch hostnames from raw data.

> Instructions (01)

Provide the engine with raw data (Ticket descriptions, CDP/LLDP outputs, or Inventory tables). As long as the text contains a Switch Hostname (e.g., PH-CISCO-CAS0101) and Port Numbers (e.g., gi1/0/3), the tool will map them.

Supported Formats: It supports standard single ports as well as comma-separated shorthand (e.g., Gi3/0/1,3,5,7).
